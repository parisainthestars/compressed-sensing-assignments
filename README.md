# Compressed Sensing Assignments

This repository contains my computer assignments for a graduate-level course on **Compressed Sensing and Sparse Recovery**.

The goal of this repo is to document the main algorithms, experiments, and results I implemented during the course, from basic sparse signal recovery to image reconstruction and sparse representation classification.

## Assignments

| Homework | Topic | Main Methods | More Details |
|---|---|---|---|
| HW1 | Sparse recovery via $\ell_0$ and $\ell_1$ minimization | Exhaustive support search, basis pursuit, linear programming | [Read HW1 README](./HW1-sparse-recovery-l0-l1/README.md) |
| HW2 | Greedy pursuit algorithms | Matching Pursuit, OMP, Subspace Pursuit, CoSaMP | [Read HW2 README](./HW2-greedy-pursuit-algorithms/README.md) |
| HW3 | Sparse recovery with different sensing matrices | Time sampling, DCT sampling, random orthonormal sensing | [Read HW3 README](./HW3-sensing-matrices-sparse-recovery/README.md) |
| HW4 | Sparse image recovery | Patch-based compressed sensing, IDCT sparsity, PSNR evaluation | [Read HW4 README](./HW4-sparse-image-recovery/README.md) |
| HW5 | Sparse representation for face recognition | SRC, OMP sparse coding, residual-based classification | [Read HW5 README](./HW5-sparse-representation-face-recognition/README.md) |
