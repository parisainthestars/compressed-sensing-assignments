# Compressed Sensing Assignments

This repository contains my computer assignments for a graduate-level course on **Compressed Sensing and Sparse Recovery**.

The assignments focus on the computational and theoretical foundations of sparse signal recovery, including underdetermined linear systems, sparsity-promoting optimization, convex relaxation, sensing matrix behavior, and empirical recovery phase transitions.

## Contents

| Homework | Topic | Main Methods |
|---|---|---|
| HW1 | Sparse Recovery via \( \ell_0 \) and \( \ell_1 \) Minimization | Exhaustive sparse search, linear programming, sensing matrix comparison |

## Goals of the Repository

The goal of this repository is to document what I implemented and learned during the course, including:

- Sparse signal models
- Recovery from underdetermined measurements
- \( \ell_0 \) minimization and its computational difficulty
- \( \ell_1 \) minimization as a convex relaxation
- Linear programming formulation of basis pursuit
- Empirical testing of recovery limits as sparsity increases
